About

Hi, I'm Arman Sheikhhosseini — a DevSecOps enthusiast passionate about building secure, automated, and scalable infrastructure.

This project is a hands-on demonstration of the end-to-end DevOps lifecycle applied to a Go web application. It covers containerization with Docker, Kubernetes deployment using both raw manifests and Helm charts, and a fully automated CI/CD pipeline with GitHub Actions.

What This Project Covers

  • Go web application development using the standard net/http package
  • Multi-stage Docker builds with a distroless base image for minimal attack surface
  • Kubernetes deployment with resource limits, services, and ingress configuration
  • Helm chart packaging for reproducible Kubernetes deployments
  • GitHub Actions CI/CD pipeline: build, test, lint, Docker push, and automated Helm tag updates

Skills Demonstrated

  • Go • Docker • Kubernetes • Helm
  • GitHub Actions • CI/CD • Docker Hub
  • Infrastructure as Code • DevSecOps practices
